## Configuração do Vue

```bash


# vue-cli
npm install -g vue-cli

# Crie o repositório e acesso o mesmo
mkdir meu-projeto
cd meu-projeto/

# Instalando as depedências
npm install

# Servidor irá rodar em localhost:8080
npm run dev


```
